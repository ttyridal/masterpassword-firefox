MasterPassword Release 3.0.0

Thank you for using MasterPassword for browsers! The easiest way to handle site-unique password on the web. If you want to understand what this is all about, please see here.

You've chosen to install a beta release of this addon. Thank you for beeing an early adopter! If you find any bugs or missing features, please report them on the issue tracker.

If you have any issues with with addon, please report them on the issue tracker.

Users of OS' key-vault

If you're using, or want to use, GNOME Keyring, KWallet, OSX Keychain or Windows Password Vault to store your master key, you'll need to download a separate helper app. Please refer to the wiki for further information.

Changes since v2.9.5

  • Implement manifest v3. This is a chrome requirement and other browser would likely follow. mv3 has no practical implcations for the use of masterpassword, but should in theory make life for rouge extensions harder.
  • Minor bug fixes
  • Public Suffix List updated
